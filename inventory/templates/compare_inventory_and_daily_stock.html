Lecture des écarts

  • Jour J : le système a enregistré au moins une mise à jour d’inventaire le même jour que le relevé gérant — les colonnes « Syst. » reprennent le dernier niveau de ce jour.
  • Cumul : aucune mise à jour inventaire ce jour-là ; on affiche le dernier niveau connu jusqu’à cette date.
  • Écart = litres déclarés (relevé) − litres système ; proche de 0 : alignement entre les deux.
  • Écart important : vérifier la saisie, les pertes éventuelles, ou des opérations non enregistrées dans l’application.
